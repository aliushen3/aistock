# 设计文档补充说明

本目录文档是对原《产业瓶颈Alpha智能选股系统整体设计方案》的**正式补充**，与原 docx **冲突时以本目录为准**。

## 核心修订摘要

1. **系统定位**：知识驱动的定性投研辅助系统
2. **量化边界**：提示分仅辅助排序，禁止自动入池与自动决策
3. **算法修订**：GNN → 规则引擎 hint_score；双图库 → 单 Neo4j
4. **知识工程**：补齐溯源、版本、冲突消解、专家校准、质量 KPI
5. **人机协同**：7 个必经人工确认节点
6. **MVP**：单赛道 AI 算力，3 个月闭环

## 阅读顺序

```
00 定位 → 01 双逻辑 → 02 知识工程 → 03 人机协同
    → 04 GraphRAG → 05 Serenity → 06 提示分
    → 07 数据 → 08 MVP → 09 评估 → 10 技术栈
```
